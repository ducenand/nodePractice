静态资源服务器

### anydoor
Tiny NodeJs Static Web Server.

### 安装

```
npm i -g anydoor
```
### 使用方法

```
anydoor # 把当前文件夹作为静态资源服务器根目录

anydoor -p  8080 # 设置端口号为 8080

anydoor -h localhost # 设置host  为localhost 

anydoor -d /usr # 设置根目录为 /usr

```
