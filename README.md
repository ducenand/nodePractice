静态资源服务器

### anydoorwhere
Tiny NodeJs Static Web Server.

### 安装

```
npm i -g anydoorwhere
```
### 使用方法

```
anydoorwhere # 把当前文件夹作为静态资源服务器根目录

anydoorwhere -p  8080 # 设置端口号为 8080

anydoorwhere -h localhost # 设置host  为localhost 

anydoorwhere -d /usr # 设置根目录为 /usr

```
